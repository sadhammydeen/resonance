## ⚡ Quick Start (5 minutes)

### 1. Open TWO terminals in VS Code

**Terminal 1 - Backend:**
```bash
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
copy .env.example .env
# Edit .env and add your OpenAI API key
python main.py
```

**Terminal 2 - Frontend:**
```bash
cd frontend
npm install
npm start
```

**OR** just double-click: `start.bat` (Windows)

### 2. Test It
- Frontend opens at: http://localhost:3000
- Upload a song (any MP3/WAV file)
- Watch it analyze WITHOUT needing sound
- See rhythm maps, emotion timeline, AI explanations

### 3. Verify It Works
- [ ] Backend starts without errors
- [ ] Frontend opens in browser
- [ ] Can upload audio file
- [ ] Analysis completes
- [ ] All visualizations display
- [ ] Explanations generate

**If something fails**: Check [SETUP.md](SETUP.md) troubleshooting section
